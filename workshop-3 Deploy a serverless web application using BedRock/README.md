# Serverless Web Application with AWS Bedrock

A serverless web application that uses AWS Bedrock for AI image generation capabilities.

## Project Structure

- `AWS-Amplify-Code/` - Frontend web application code
- `lambda_function.py` - AWS Lambda function for backend processing
- `lambda-iam-role-cfn.yaml` -  Lambda IAM role
- `car.jpg` - Sample image
- `generated_image_1 output.png` - Generated image output

## Components

- **Frontend**: Web interface hosted on AWS Amplify
- **Backend**: AWS Lambda function with Bedrock integration


## Getting Started


1. Deploy the Lambda function
2. Host the web application using AWS Amplify
3. Configure the frontend to connect to your Lambda function

## AWS Services Used

- AWS Bedrock
- AWS Lambda
- AWS Amplify
